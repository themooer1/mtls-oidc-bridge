# openid-tls-connector-2

To install dependencies:

```bash
bun install
```

To run unit tests:

```bash
bun run test
```

To run:

```bash
bun run src/main.ts
```

For Docker-backed smoke, LDAP integration, and OpenID Foundation conformance tests, see [`integration tests/README.md`](integration%20tests/README.md).

This project was created using `bun init` in bun v1.3.14. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
